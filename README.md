1. ¿Qué es la Inteligencia Artificial (IA)?La IA es el campo de la informática que busca crear sistemas capaces de realizar tareas que, normalmente, requerirían inteligencia humana. Esto incluye razonar, resolver problemas, comprender el lenguaje natural o reconocer objetos en una imagen.IA Débil (o Estrecha): Diseñada para una tarea específica (como Alexa, los filtros de spam o el ajedrez). Es lo que tenemos hoy.IA Fuerte (AGI): Una máquina que podría realizar cualquier tarea intelectual que un humano puede hacer. (Aún es teórica).

2. ¿Qué es el Machine Learning (ML)?El Machine Learning (Aprendizaje Automático) es una rama de la IA que se basa en la idea de que los sistemas pueden aprender de los datos, identificar patrones y tomar decisiones con una intervención humana mínima.En lugar de programar reglas fijas (si pasa A, haz B), "entrenamos" al algoritmo con miles de ejemplos para que él mismo descubra la lógica.
  
4. 3. Tipos de Machine LearningDependiendo de cómo "estudie" la máquina y qué tipo de datos reciba, existen principalmente cuatro tipos de aprendizaje:
      
      A. Aprendizaje SupervisadoEs como tener a un profesor corrigiendo un examen. El algoritmo recibe datos "etiquetados" (con la respuesta correcta incluida).
           Ejemplo: Le das 10,000 fotos de gatos etiquetadas como "Gato" y 10,000 de perros etiquetadas como "Perro". El modelo aprende a identificar las diferencias.
           Uso común: Detección de spam, predicción de precios de casas.
      
      B. Aprendizaje No SupervisadoAquí no hay etiquetas ni profesor. El algoritmo recibe datos brutos y debe encontrar estructuras o grupos por sí solo.
           Ejemplo: Le das datos de millones de clientes de un supermercado y el modelo los agrupa automáticamente por "hábitos de compra" (segmentación).
           Uso común: Sistemas de recomendación (Netflix/Spotify), detección de anomalías bancarias.
      
      C. Aprendizaje por Refuerzo (Reinforcement Learning)Se basa en el sistema de recompensa y castigo. El algoritmo (agente) interactúa con un entorno y aprende mediante ensayo y error para maximizar una puntuación.
           Ejemplo: Un software aprendiendo a jugar Mario Bros. Si muere, recibe un "castigo"; si avanza o llega a la meta, recibe una "recompensa".
           Uso común: Robótica, coches autónomos, videojuegos.
      
      D. Aprendizaje Semi-supervisadoEs una mezcla de los dos primeros. Se usa cuando tienes una cantidad pequeña de datos etiquetados y una cantidad enorme de datos sin etiquetar (porque etiquetar a mano es caro y lento).
           Uso común: Clasificación de imágenes médicas.

      resumen de mark down
      
      Markdown es un lenguaje de marcado ligero que permite aplicar formato a un texto mediante símbolos sencillos de teclado. Es el estándar favorito de programadores y escritores porque es fácil de leer, incluso si no está "renderizado".

Aquí tienes los comandos esenciales que necesitas conocer:

1. Títulos y Estructura
Se utilizan almohadillas (#) al inicio de la línea. A más almohadillas, menor es el tamaño del título.

# Título Principal (H1)

## Subtítulo (H2)

### Sección (H3)

2. Formato de Texto
Para dar énfasis, rodeamos las palabras con asteriscos o guiones bajos:

Negrita: **texto** o __texto__

Cursiva: *texto* o _texto_

~~Tachado~~: ~~texto~~

3. Listas
Puedes crear listas de forma muy natural:

Listas desordenadas:

* Elemento A

- Elemento B

Listas ordenadas:

1. Primer paso

2. Segundo paso

4. Enlaces e Imágenes
La sintaxis es similar, pero las imágenes llevan un signo de exclamación al principio.

Link: [Texto del enlace](https://google.com)

Imagen: ![Descripción](URL-de-la-imagen.jpg)

5. Código y Citas
Ideal para resaltar términos técnicos o frases célebres:

Código en línea: Usar comillas simples invertidas: `Variable_X`.

Bloque de código: Usar tres comillas invertidas (triple backticks) arriba y abajo.

Cita: Usar el signo mayor que: > Esto es una cita textual.

Tip Pro: Markdown es el lenguaje que estoy usando ahora mismo para responderte. Si te fijas, las negritas y los títulos ayudan a que la lectura sea mucho más ágil.
